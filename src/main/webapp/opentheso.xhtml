C'est un gestionnaire de thesaurus multilingue, développé par la plateforme Technologique TRD (Têtes des Réseaux Documentaires) située à la MOM
en partenariat avec le GDS-FRANTIQ

Le développement de la version 3 a bénéficié d’une participation financière du Consortium MASA (Mémoire des archéologues et des Sites Archéologiques) de la TGIR Corpus,
ce financement a permis de reprendre l’ancienne version d’Opentheso et de produire une version FullWeb qui respecte la nouvelle norme des thésaurus internationale.


Chef de Projet : Miled Rousset
Développement : Carole Bonfré et Miled Rousset
Partenariat, test et expertise : Les équipes du réseau Frantiq

Le développement a été réalisé avec les technologies suivantes :
  • PostgreSQL pour la base des données
  • Java pour le module API et module métier
  • JSF2 et PrimeFaces pour la partie graphique


Opentheso s'appuie sur le projet Arkéo de la MOM pour générer des identifiants type ARK

Modules complémentaires :
  • Hikari
  • Weka
  • Skosapi
  • Kj-jzkit
